# MultipleModules

Konzolos alkamazás (multiple modules)
Nyilván kell tartni a versenyzők adatát (név, pontszám, születési év) 

Műveleteket kell tudjon: új eredmény felvétel, legjobb, legrosszabb versőnyző kiírása, átlag pont szám
 
Használjon stream apit!

És tudjon keresni eredményeket név alapján (pl Be írom hogy "K" és minden "K"betűvel kezdődő nevet ki írja).
